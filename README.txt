# Poker

A simple poker game simulater where you can draw cards and see what the highest ranking hand matches.

You can find the github repo at https://github.com/Tjaart-Booyens/Poker.

Please check the repo for the source code and a more comprehensive README for the project.

## Running the JAR

You can run the JAR file if you have JAVA by running the following command:

~$ java -jar poker-1.0-jar-with-dependencies.jar
